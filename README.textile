h2. Hypermedia

We are ready to go, hypermedia supported:

<pre>
from restfulie import Restfulie

# using restfulie as an http api:
>>> response = Restfulie.accepts('application/xml').at('http://localhost:8080/items').get()
>>> print response.body
&lt;items>
    &lt;item>
        &lt;name>Car&lt;/name>
        &lt;price>32000.00&lt;/price>
    &lt;/item>
    &lt;item>
        &lt;name>House&lt;/name>
        &lt;price>231000.00&lt;/price>
    &lt;/item>
&lt;/items>

>>> print response.code
200

# unmarshalling the items response
>>> r = response.resource()
>>> print len(r.item)
2
>>> print len(r.item[0].name)
Car

# navigating through hypermedia
>>> item = { 'name': 'New product', 'price': 30 }
>>> result = items.link("self").follow().post(item)
>>> print result.code
200

</pre>

This is it. Adding hypermedia capabilities and following links. Now its time to use it in the <b>right way</b>.

h2. Installing Restfulie

On project root, run:

@$ python setup.py install@

h2. Installing Restfulie for development

On project root, run:

@$ python setup.py develop@


h2. Running tests

On project root, run:

@$python setup.py nosetests@
